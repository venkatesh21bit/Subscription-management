# Vendor ERP Backend - Test Suite Documentation

Complete test suite documentation in [tests/README.md](tests/README.md)

## Quick Test Commands

```bash
# Run all tests
python run_tests.py

# Run specific categories
python run_tests.py --api           # API tests
python run_tests.py --unit          # Unit tests
python run_tests.py --integration   # Integration tests

# Run with coverage
python run_tests.py --coverage

# Run in parallel (faster)
python run_tests.py --parallel
```

See [tests/README.md](tests/README.md) for complete documentation.
